Bienvenido A Mi Página Con Estilo Embebidos.

Este es un ejemplo del uso estilo embebido con html.

Estilos embebidos en las etiquetas html.

Usar estilo CSS embebido en las etiquetas html es el peor y menos utilizados de los métodos
para agregar reglas de estilos. No es nada recomendable por la dificultad que implica realizar un cambio
a posteriori.

Para insertar un estilo CSS en un elemento se utiliza la propiedad style de html
que en ningún momento se debe confundir con la etiqueta
<style>
que se introducen en la cabecera del documento.
