Ejemplos de estilo sin CSS

Ejemplo de estilo interno (CSS en el <head>).

En este ejemplo muestra como aplicar estilos mediante CSS interno.
es decir, escribiendo las reglas de estilos dentro de la etiquetas <style>
en la sección <heat> del documento html.
Esta forma permite centralizar los estilos de una página y mantener separado
el contenido (html) de la presentación (CSS), facilitando el mantenimiento y la
organización.

Ejemplo del atributo style (CSS en linea)

Los estilo se aplican directamente en una carpeta html, mediante el atributo style.
Se usa para ejemplos rápidos o pruebas, pero no se recomienda
en el diseño profesional, ya que mezcla estructura y presentación.
